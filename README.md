# Furry Hub – MM2 Roblox Script

**Version:** 2.5  
**Author:** FertonFox  

---

## Description
Furry Hub is a feature-rich Roblox Murder Mystery 2 (MM2) hub designed to enhance gameplay with automation, teleportation, optimization, ESP, and GUI customization. It includes a wide range of tools such as autofarm, player stats, role notifications, item spawning, and anti-steal protections.

This script is specifically built to run on MM2 (Place ID: 142823291).

---

## Features

### Main Features
- **Autofarm:** Automatically farm Coins, Eggs, or Candy.
- **Player Stats:** Display account information, equipped items, and XP.
- **Role Notifications:** Identify the Murderer and Sheriff during rounds.
- **ESP:** Highlight players and display names above them.
- **Item Spawner:** Spawn popular items like XenoKnife, XenoGun, and more.
- **Anti-Steal:** Blocks background trade functions.
- **Optimization Modes:** Low, Mid, and Ultra optimization for improved performance.
- **Teleportation:** Teleport to players, roles, or specific map locations.
- **GUI Themes:** Multiple themes including Halloween, Crimson, Ocean, and Dark.
- **Chat Integration:** Automatically sends messages in Roblox chat.

### Additional Tools
- **Keybinds:** Quick access to GUI with a customizable key.
- **Rejoin & Job Join:** Easily rejoin the current server or join a friend's game via Job ID.
- **Anti-AFK:** Prevents idle kicks.
- **Notifications:** Alerts for autofarm, teleports, and role detection.

## Usage

1. Open the **Furry Hub GUI** using the keybind (default: `G`).
2. Navigate between sections like:
   - **Main:** Player stats and autofarm toggles.
   - **Farms:** Adjust farm type and speed.
   - **Optimization:** Select your device's optimization mode.
   - **Teleports:** Teleport to players, roles, or map locations.
   - **ESP:** Enable visual player highlights.
   - **Item Spawner:** Spawn your favorite weapons and items.
   - **Settings:** Customize GUI themes and transparency.
3. Toggle features on/off as needed.
